## How to Use

### 1. Create a Status Page
```
1. Go to: http://localhost:3000/dashboard/status-pages
2. Click "CREATE_STATUS_PAGE"
3. Fill in name, slug, description, color
4. Click "CREATE"
```

### 2. Edit Status Page
```
1. Click on a status page card
2. Click "EDIT_SETTINGS" button
3. Update name, description, logo, color, CSS
4. Click "SAVE_CHANGES"
```

### 3. View Public Page
```
1. From status page detail, click "VIEW_PUBLIC_PAGE"
2. Or visit: http://localhost:3000/status/your-slug
3. Share this URL with your users
```
